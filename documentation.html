Documentation

We offer several options to use MEVA:
  • Download the executable with the included case study data and use it for exploring the given data sets.
  • Install Unity and download the MEVA Unity project. Explore the project, import your own data and implement your own assets.
  • Download the simulation data (netCDF input data) and use the NCL scripts and ParaView scripts to generate the geometries which you can use in Unity.
