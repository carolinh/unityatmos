MEVA

multifaceted environmental data visualization application

Download this project as a .zip file Download this project as a tar.gz file

Navigation

  • Show/hide menu: right mouse button
  • Swap menu (objects/time): middle mouse button
  • Change camera (default, split screen vertical and horizontal): space
  • Rotate scene: arrow keys
  • Zoom scene: page up/down
